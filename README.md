

# KOPA: Lakeshore Development

A premium digital experience platform designed for the KOPA destination in Koregaon Park, Pune. This project serves as a sophisticated, high-end presentation deck, showcasing architectural excellence, consumer demographic data, and venue specifications through an immersive, gallery-style interface.

## Key Features

* **Cinematic Arrival:** An immersive, full-screen video intro with bespoke motion transitions.
* **Gallery-Style Layouts:** Editorial-focused, split-screen slides that treat data and photography as curated museum installations.
* **Premium Motion:** Powered by `framer-motion` for fluid, high-end animations that provide a tactile, expensive feel.
* **Responsive Architecture:** Designed with Tailwind CSS for pixel-perfect clarity across devices.
* **Data-Driven:** Dynamic presentation of catchment demographics and retail statistics.

## 🛠 Tech Stack

* **Framework:** React (Vite)
* **Animation:** Framer Motion
* **Styling:** Tailwind CSS
* **Utilities:** `react-intersection-observer`, `useCountUp`
* **sora:** to create ai generated videos 

## Getting Started

### Prerequisites

* Node.js (v18+)
* npm or yarn
* sora

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/kopa-experience.git
cd kopa-experience


2.  **Install dependencies:**
    ```bash
npm install

3. **Run the development server:**
```bash

npm run dev

```

## 🏗 Project Structure

*   `/public`: Static assets (video files, icons, logos).
*   `/src/components`: Reusable UI components (stats, typography, modular layouts).
*   `/src/data`: Centralized content management for demographics and venue specs.
*   `/src/hooks`: Custom animation and count-up logic.

##  Design Philosophy
The KOPA experience is built on the **"Gallery Museum" aesthetic**:
*   **Whitespace:** Prioritizing space to let content breathe.
*   **Typographic Hierarchy:** Refined, minimal headers paired with gold accent (`#c4973a`) structural lines.
*   **Cinematic Transitions:** Deliberate, ease-based motion that feels intentional rather than mechanical.
